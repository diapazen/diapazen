viewPoll (   $pollUrl)

Affichage d'un sondage en récupèrant les informations stockées dans la base de données au moyen d’une requête sql Select sur la vue dpz_view_users_join_polls. On traite le résultat obtenu en récupérant les informations de chaque choix du sondage ainsi que les informations de chaque résultat des choix du sondage. Les résultats obtenus sont ensuite transférés dans un tableau et on calcule le pourcentage de chaque choix du sondage. Le tableau est donc complété avec ces mêmes pourcentages.

type$pollUrlurl du sondage
